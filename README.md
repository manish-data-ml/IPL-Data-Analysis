# IPL Data Analysis Project

## 📌 Objective
This project analyzes IPL dataset to extract insights about team performance, player performance, and match outcomes.

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Most winning team
- Toss decision analysis
- Top scorers
- Best bowlers
- Highest win margin
- Highest individual score

## 📈 Key Insights
- Gujarat is the most successful team in this dataset.
- Teams prefer fielding after winning the toss.
- Jos Buttler is the top scorer.
- Yuzvendra Chahal is the top bowler.
